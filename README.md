# SARA 看CB玩現股（WEB 原型）

**線上預覽：https://cmoneyjoseph.github.io/saraCB/**

SARA 的可轉債策略 WEB 工具原型。核心概念：**CB 是訊號，現股才是主角**——股債乖離收斂進 ±3%，多半代表短線上現股即將或正在發動。

## 策略

| 策略 | 邏輯 |
|---|---|
| 看CB玩現股（主） | 股債乖離率 ≤10% ＋ 未轉換比率 ≥80% ＋ CB收盤 ≥100，乖離進 ±3% 標「發動」 |
| 發動雷達 | 股債乖離率進入 ±3%，依心動指標排序（優先看大橘） |
| 型態共振 | 現股符合 SARA 型態（N字勾／小碎步／有底撐／穿心箭）＋ 股債乖離 ≤10% |
| 全部總覽 | 可轉債完整清單 |

## 計算定義

- 轉換價值 = 100 ÷ 轉換價 × 股價
- 股債乖離率 = (CB價 − 轉換價值) ÷ 轉換價值
- 心動指標：−10～10 分級（<0 藍綠、0–3 白、4–6 淺橘、7–10 橘紅）

## 互動

- 點策略 chip 切換濾網；點任一檔開「發動判讀」——乖離儀表（±3% 金色發動區）＋股債連動＋現股觀察
- 手機自動切換卡片版

## 原型資料說明

- 行情、轉換價、股債乖離為 **2026/07/02–03 真實盤後樣本**
- 未轉換比率、心動指標、現股型態標籤為**示意值**，僅供架構與互動討論，勿作投資參考
