# SARA CB 策略雷達（WEB 原型）

**線上預覽：https://cmoneyjoseph.github.io/saraCB/**
**策略邏輯與使用說明：https://cmoneyjoseph.github.io/saraCB/guide.html**（站內「操作說明」按鈕會開啟快速導覽，首次進站自動顯示）

SARA 的可轉債交易 WEB 工具原型。核心概念：**現股給訊號，CB 下單**——現股心動、型態到位時，改用可轉債進場（上有跟漲、下有債底）；小資金想放大，用 CBAS 槓桿。

## 策略（交易標的都是 CB / CBAS）

| 策略 | 邏輯 |
|---|---|
| 心動CB（主） | 現股心動 ≥4 或型態到位 ＋ 股債乖離 ≤10% ＋ CB 有成交 → 單下在 CB |
| 發動雷達 | 股債乖離進 ±3%，CB 與現股 1:1 連動 → 買 CB ＝ 買現股＋債底保險 |
| 債底安全網 | CB ≤105 ＋ 賣回報酬率 ≥0 → 跌有保底、漲跟股票，進可攻退可守 |
| CBAS槓桿 | 權利金 <20 ＋ 槓桿 >5 倍 → 小額權利金撬動整檔 CB，搭配高心動訊號 |
| 全部總覽 | 可轉債完整清單 |

## 計算定義

- 轉換價值 = 100 ÷ 轉換價 × 股價
- 股債乖離率 = (CB價 − 轉換價值) ÷ 轉換價值
- 賣回報酬率 = (賣回價 − CB價) ÷ CB價
- CBAS 權利金 = (CB − 100) ＋ 百元報價（折現率預設 1.5%、手續費 0.3%，明細內可調）
- 心動指標：−10～10 分級（<0 藍綠、0–3 白、4–6 淺橘、7–10 橘紅，優先大橘）

## 互動

- 點策略 chip 切換濾網；點任一檔開明細——**CB 進場判讀**（乖離儀表＋折價/溢價白話解讀）＋ **CBAS 槓桿試算**（折現率可調）＋ 債底條款 ＋ 現股訊號
- 手機自動切換卡片版

## 原型資料說明

- CB／現股行情、轉換價、乖離、賣回條件為 **2026/07/02–03 真實盤後樣本**
- 未轉換比率、心動指標、現股型態標籤為**示意值**，僅供架構與互動討論，勿作投資參考
