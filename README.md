# SARA 可轉債策略雷達（WEB 原型）

**線上預覽：https://cmoneyjoseph.github.io/saraCB/**

SARA 品牌的可轉債（CB）選債 WEB 工具原型。

## 架構概念

**一張可轉債總覽主表 ＋ 策略 preset 切換**：所有策略共用同一份資料，只切換篩選條件、欄位與排序。

| 策略（暫定名） | 篩選邏輯 |
|---|---|
| 創高衝鋒 | 創20/60/120日新高＋折溢價<40%＋(殖利率>6% 或 營收成長>15%) |
| 賣回收益 | 賣回報酬率>0＋EPS>0＋負債比<50% |
| 折價套利 | 折溢價<0＋有成交＋融券可空 |
| CBAS優勢 | 權利金<20＋槓桿>20倍 |
| 低本益雷達 | CB<110＋本益比<20＋(成長>30% 或 殖利率>6%) |
| 主力集保 | 集保佔發行比>50% |

## 核心公式

- 理論價 = 100 ÷ 轉換價 × 標的收盤價
- 折溢價 = (CB價 − 理論價) ÷ 理論價
- 賣回報酬率 = (賣回價 − CB價) ÷ CB價
- CBAS 權利金 = (CB − 100) ＋ 百元報價（折現率預設 1.5%、手續費 0.3%，明細內可調）

## 原型資料說明

- 行情、理論價、折溢價為 **2026/07/02–03 真實盤後樣本**
- 賣回條款、集保比率、財務預估等欄位**部分為示意值**，僅供架構與互動討論，勿作投資參考
